# irop 施術事例集 要件定義

## アクセス・認証

- **閲覧**: URLを知っていれば誰でもアクセス可能（ログイン不要）
- **事例登録**: 招待制（ブランドマネージャーが招待した美容師のみ登録可能）
- **承認フロー**: 美容師が登録した事例は、ブランドマネージャーが確認・承認してから公開される
  - → 登録画面、管理画面（承認機能）、招待機能が必要

## 登録の利用シーン

- サロンで施術直後にスマホから撮影・登録するケース
- 後からまとめて登録するケース
- → どちらにも対応できるUI（スマホでの登録しやすさが重要）

## 規模感

- 初期リリース: 10件程度からスタート
- 徐々に増やしていく運用

## データ項目の詳細

### Before（施術前）情報

| 項目 | 入力形式 | 詳細 |
|------|---------|------|
| Before写真 | 画像アップロード | 施術前の髪の写真（任意。未登録時は「No Image」プレースホルダー表示） |
| 髪の明度 | 数値入力（レベル） | 登録時は数値、検索時は「5-7」など範囲で検索可能に |
| 髪の色相 | 選択式（16種）＋自由記述 | irop診断ロジック（PCCS準拠）に基づく16色相グループから選択。将来は写真から自動推定したい |
| 白髪率 | 選択式 | 「10%未満」「10-20%」「30-50%」「50-70%」「70%以上」 |
| 施術履歴 | 選択肢＋自由記述 | 主な施術はチェックボックス、補足を自由記述 |

#### 施術履歴の選択肢（候補）
- 縮毛矯正
- パーマ
- ブリーチ
- 通常カラー（アルカリカラー）
- その他（自由記述）
※選択肢は今後追加・調整の可能性あり

### After（施術後）情報

| 項目 | 入力形式 | 詳細 |
|------|---------|------|
| After写真 | 画像アップロード | 施術後の髪の写真 |
| 使用シャンプー | カラー選択（5色）＋ミックス対応 | 単品 or 複数色＋配合比率。使用しない場合もある |
| 使用トリートメント | カラー選択（5色）＋ミックス対応 | 単品 or 複数色＋配合比率。使用しない場合もある |
| 放置時間 | 数値入力（分） | 例: 5分、10分 |

### フィルター検索項目

| フィルター項目 | 検索方式 |
|-------------|---------|
| 髪の明度 | 範囲指定（例: 5-7レベル） |
| 髪の色相 | 選択式（16色相グループから選択、複数選択可） |
| 白髪率 | 選択式（「10%未満」「10-20%」「30-50%」「50-70%」「70%以上」） |
| 施術履歴 | チェックボックス（複数選択可） |
| 使用したiropカラー | チェックボックス（5色から選択） |

## デザイン

- iropのブランドカラー・トーンに合わせたデザイン（詳細: `docs/knowledge/brand-design.md`）
- モバイルファースト（スマートフォンでの閲覧・操作を最優先）
- フォント: Typo Round（英数字指定、埋め込み不可時はComfortaa代用）、Zen Kaku Gothic New（和文、および日本語混在・可読性重視時の英字）※ブランドガイドラインに準拠
- フォントファイル: `design/英字typo_roundフォント/`（Thin/Light/Regular/Bold + Italic）
- トーン: シンプルで上質、クリーン

## 将来の拡張（Phase 2以降）

- 写真からの髪の色相自動推定（AI活用）

## ホスティング・ドメイン

- **既存サイト**: irop.jp は ecforce（ECプラットフォーム）で運営
- **ドメイン管理**: Xserver でドメインを取得
- **希望**: irop.jp の一部として公開したい（例: salon.irop.jp）
  - → Xserverでサブドメインを設定し、別のホスティング先に向ける構成が考えられる
