# ヘアカラー関連の基礎知識

## 塩基性カラーとは

- ヘアカラーの一種で、一時的な染毛剤に分類される
- 髪の表面に色素を付着させる仕組み（髪の内部に浸透するアルカリカラーとは異なる）
- 髪へのダメージが少ない
- シャンプーのたびに少しずつ色が落ちる（一時的）

## 美容師が気にする髪の情報

施術事例を理解するために、美容師が知りたい基礎情報:

- **明度（レベル）**: 髪の明るさ。数字が大きいほど明るい（例: 3=黒髪、13=金髪に近い）
- **色相**: 髪の色味。iropではPCCS（日本色研配色体系）をベースにした16グループで分類（詳細は下記）
- **彩度**: 色の鮮やかさ
- **白髪の有無・白髪率**: 白髪がどのくらい混じっているか（%で表現）
- **施術履歴**: 過去にどんな施術を受けているか
  - 例: 縮毛矯正、パーマ、ブリーチ、通常カラーなど
  - これにより薬剤の入り方や色の出方が変わる

### トーンマップ（明度×彩度）

診断ロジックでは、色相に加えてPCCSのトーン分類も使用:
- 縦軸: 明度（W=ホワイト〜Bk=ブラック）
- 横軸: 彩度（低い←→高い）
- 主なトーン: p(ペール), lt(ライト), b(ブライト), sf(ソフト), s(ストロング), v(ビビッド), d(ダル), dp(ディープ), dk(ダーク) など

※ 施術事例集の初期版では色相グループ（16種）の選択のみを実装し、トーン分類は将来の拡張候補とする

## iropの施術に関するパラメータ

- **使用アイテム**: iropのどのカラーを使ったか
- **ミックス割合**: 複数色を混ぜた場合の配合比率
- **放置時間**: 塗布後にどのくらい時間を置いたか
