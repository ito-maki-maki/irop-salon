# irop 施術事例集 — マイルストーン計画

作成日: 2026-04-09
改訂日: 2026-04-10（静的HTML方式に変更）

## 方針

- 閲覧機能（ログイン不要）を最優先とし、投稿・管理機能を後続で構築する
- 各マイルストーン完了時に動作するものが確認できるように設計
- **静的HTMLファイルで構築**（ビルド不要、ブラウザで直接開いて確認）
- DB・認証・ホスティングは画面構築後に決定
